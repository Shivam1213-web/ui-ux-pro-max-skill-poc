# FinFlow Dashboard

FinFlow is the prototype fintech dashboard built for the [Fintech UI POC](../README.md). For project overview, objectives, tools, repository structure, and next steps for Satva, see the [repository README](../README.md).

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Use the sidebar to navigate: Dashboard, Accounts, Payments, Transactions, Analytics, AI Insights, Settings.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm start` | Run production server |
| `npm run lint` | Run ESLint |

## Tech Stack

- **Next.js 16** (App Router) · **React 19** · **Tailwind CSS v4** · **Recharts** · **TypeScript**
